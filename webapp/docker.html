¿Qué es Docker?

Contenedor Docker Ubuntu

Puede ser útil confirmar los cambios o la configuración del archivo de un contenedor en una nueva imagen. Esto le permite depurar un contenedor ejecutando un shell interactivo o exportar un conjunto de datos de trabajo a otro servidor. Generalmente, es mejor usar Dockerfiles para administrar sus imágenes de una manera documentada y mantenible.

La operación de confirmación no incluirá ningún dato contenido en los volúmenes montados dentro del contenedor.

De forma predeterminada, el contenedor que se está confirmando y sus procesos se pausarán mientras se confirma la imagen. Esto reduce la probabilidad de encontrar corrupción de datos durante el proceso de creación de la confirmación. Si no desea este comportamiento, establezca la --pause opción en falso.
